default commands

homes

/sethome [name]: sets a spawn for you. pretty much personal warps
/home [name]: go to one of your homes (you don't need a name if you have only one home)
/renamehome [old name] [new name]: rename one of your homes
/delhome [name]: delete one of your homes

tp requests

/tpa [player]: request to teleport to a player. stand still for 5 seconds if they accept.
/tpahere [player]: request a player to tp to you. they will need to stand still.
/tpaccept: accept an incoming tp request.
/tpdeny: deny an incoming tp request.
/tpcancel: cancel any of your outgoing tp requests.

misc

/msg [player]: privately message a player.
/back: return to the spot you died.
/hat: puts the item in your hand as a hat. just for fun.
/rtp: teleport somewhere random.

commands per rank

testers

everything from default
/feed: refill selected player's hunger
/heal: regen player to max hearts
/repair [all|hand]: set durability of item to max
/echest: open your permanent ender chest
/gamemode [0|1|2|3]: change your gamemode
/warp: tp to an existing warp
/warplist: view all existing warps

helpers

everything from default and tester
/createwarp [warp name]: create a warp
/tp [player]: teleport to a player without a request or wait
+ access to use tester cmds on other players

elites

everything from the other ranks
/kick [player]: kick a player from the server
/tempban [player] [time]: ban a player for a set amount of time
/mute [player]: shut someone up
/cell [player] : send someone to the slammer
/enchant [enchant] [level]: enchant an item in your hand, max lvl 32,768
/summon [item]: summon anything, anywhere
/give [item] [amount]: give anybody any item, even not from the creative ui

do not use brackets in commands!
